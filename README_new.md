# Проект по автоматизации тестирования сервиса «Яндекс.Самокат»

Проект содержит автотесты на Java для веб-приложения [Яндекс.Самокат](https://qa-scooter.praktikum-services.ru).
Проект реализован с использованием паттерна Page Object Model (POM) и параметризации.

## Технологии и инструменты

| Технология    | Версия |
|---------------|--------|
| Java          | 11     |
| JUnit       | 4.13.2 |
| Maven       | 3.9.14 |
| Selenium WebDriver | 4.32.0 |
| WebDriverManager| 5.5.3  |

### Предварительные требования
Браузеры Google Chrome и FireFox.

### Запуск тестов
Для запуска всех тестов из терминала используй команду:

```shell
mvn clean test
